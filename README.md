# System Design Archive

시스템 디자인 학습 과정을 기록하고 아카이빙하는 저장소입니다.

## 구조

```
├── diagnosis/          # 수준 진단 & 격차 분석
├── curriculum/         # 학습 로드맵 & 주간 계획
├── system-design/      # 시스템 디자인 챌린지 & 리뷰
│   ├── challenges/
│   └── reviews/
├── theory/             # CS 이론 (네트워킹, DB, OS, 알고리즘)
│   ├── networking/
│   ├── database/
│   ├── os/
│   └── algorithms/
├── career/             # 이력서 & 모의 면접
└── code-review/        # 코드 리뷰 기록
```

## 학습 방법론

**역학습(Reverse Learning)** — 시스템 디자인 문제를 먼저 풀고, 드러나는 CS 기초 빈틈을 채우는 방식으로 학습합니다.

> 시스템 디자인 챌린지 → 부족한 이론 발견 → 이론 보충 → 장기 기억 정착
